# 🍽️ PillaMesa - Sistema de Gestión de Reservas para Restaurantes

[![Java](https://img.shields.io/badge/Java-20-blue)](https://www.oracle.com/java/)
[![SQLite](https://img.shields.io/badge/SQLite-Database-lightgrey)](https://www.sqlite.org/)
[![JasperReports](https://img.shields.io/badge/JasperReports-6.18.1-orange)](https://community.jaspersoft.com/)
[![Maven](https://img.shields.io/badge/Maven-Build-red)](https://maven.apache.org/)
[![JavaHelp](https://img.shields.io/badge/JavaHelp-2.0.05-green)](https://javahelp.dev.java.net/)

**PillaMesa** es una aplicación de escritorio desarrollada en **Java 20** con **Swing**, diseñada para la gestión eficiente de reservas en restaurantes. 

Con esta aplicación, los administradores y camareros pueden gestionar clientes, mesas y reservas de manera sencilla, así como generar informes detallados en **PDF y HTML**.

---

## 🚀 **Características Principales**
✔ **Gestión de clientes:** Alta, baja y modificación de clientes.  
✔ **Gestión de mesas:** Control de disponibilidad y capacidad de mesas.  
✔ **Reservas:** Creación, cancelación y modificación de reservas.  
✔ **Gestión de usuarios:** Control de acceso para **administradores** y **camareros**.  
✔ **Reportes:** Generación de informes en **PDF y HTML** con JasperReports.  
✔ **Interfaz gráfica moderna** basada en **Swing**.  
✔ **Uso de JavaHelp** para documentación interactiva dentro de la aplicación.  
✔ **Base de datos SQLite**, no requiere configuración adicional.  

---

## 🛠️ **Tecnologías Usadas**
- **Java 20** - Lenguaje principal de desarrollo.
- **Swing** - Interfaz gráfica de usuario.
- **SQLite** - Base de datos embebida.
- **JasperReports** - Generación de informes en PDF y HTML.
- **Maven** - Gestión de dependencias y construcción del proyecto.
- **JavaHelp** - Sistema de ayuda interactiva para los usuarios.
