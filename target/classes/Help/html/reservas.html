📅 Gestión de Reservas

Esta es la ventana principal de PillaMesa, donde puedes ver todas las reservas registradas en el sistema.

🔍 Funcionalidades

  • Consultar la lista de reservas.
  • Aplicar filtros para buscar reservas por cliente, mesa, fecha y hora.
  • Actualizar automáticamente las reservas de acuerdo a la fecha actual.

📌 Accesos Rápidos

Desde esta ventana puedes navegar a otras secciones como:

  • Crear Reserva
  • Gestión de Usuarios (Solo Encargados)
  • Gestión de Clientes
  • Gestión de Mesas
  • Generación de Informes
