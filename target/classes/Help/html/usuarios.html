👥 Gestión de Usuarios

Solo los Encargados pueden acceder a esta sección para gestionar los usuarios del sistema.

📌 Funcionalidades

  • Crear nuevos usuarios con roles de Encargado o Camarero.
  • Editar los datos de los usuarios existentes.
  • Eliminar usuarios que ya no forman parte del equipo.

📌 Roles de Usuario

Existen dos tipos de usuarios en el sistema:

  • Encargado: Acceso completo a todas las funcionalidades.
  • Camarero: Solo puede ver y gestionar reservas.

⚠️ Nota: No se puede eliminar el usuario actualmente en sesión.
